# MVP_RELEASE.md

## Этап 9 — подготовка MVP к демонстрации

## 1) Предрелизная проверка

Выполнить в PowerShell:

```powershell
npm install
npm run typecheck
npm run build:web
```

Ожидаемый результат:
- `typecheck` без ошибок;
- веб-экспорт проходит успешно, артефакты формируются в `dist/`.

## 2) Проверка основных сценариев

Использовать чек-лист из `TEST_STAGE8.md`:
- места;
- поездки и маршрут;
- следующее место;
- перезапуск приложения и сохранность данных.

## 3) Демонстрация MVP

### Вариант A: мобильная демонстрация через Expo Go

```powershell
npm run start -- --clear
```

Далее:
- открыть приложение Expo Go на телефоне;
- отсканировать QR-код;
- показать основные экраны и полный сценарий.

### Вариант B: веб-демонстрация

```powershell
npm run web
```

Или показать экспортированную версию из `dist/` после `npm run build:web`.

## 4) Контрольный список релизной готовности

- [ ] Приложение запускается на Android/Web без падений.
- [ ] Типизация проходит (`npm run typecheck`).
- [ ] Сценарий из `TEST_STAGE8.md` пройден полностью.
- [ ] Локальное хранение данных и фото работает.
- [ ] Экран `Следующее место` корректно определяет следующий пункт.
- [ ] Документация актуальна (`README.md`, `PLAN.md`, `TEST_STAGE8.md`, `MVP_RELEASE.md`).
